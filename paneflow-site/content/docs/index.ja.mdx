Paneflow は、AI エージェント CLI ワークフロー向けに構築された
クロスプラットフォームのターミナル マルチプレクサです -
Claude Code、Codex、OpenCode、そしてネイティブシェルを話すあらゆる
エージェントに対応します。ペインの分割、永続セッション、開発
サーバー ポートの検出、設定不要のテーマ。これらすべてが、200 ms
未満で起動するネイティブ GPU 加速アプリケーションに統合されて
います。

<Callout type="info">
Paneflow は、**macOS**（公証済み `.dmg`、Apple Silicon）と
**Linux**（`.deb`、AppImage、tarball、x86_64 および ARM64）向けに
署名済みバイナリを配布しています。上のボタンは各 OS に推奨される
形式を取得します。下の OS 別ガイドはその他すべてのパスをカバー
します。
</Callout>

## 内部で動くもの

- **純粋な Rust** + [GPUI](https://www.gpui.rs/) - Zed が動作する同じフレームワーク。
- **ネイティブ VT エミュレーション**、`alacritty_terminal`（crates.io 0.26）経由。
- **ネイティブ PTY**、`portable-pty` 経由 - Linux と macOS で同じコード パス。
- **JSON-RPC IPC**、`$XDG_RUNTIME_DIR/paneflow/paneflow.sock`、ヘッドレス スクリプト用。
- **コールド スタート 200 ms 未満**、レイテンシ プローブでキーストロークからピクセルまで 4 ms 未満。

## システム要件

| プラットフォーム | 最小要件 | GPU |
| --- | --- | --- |
| Linux | glibc 2.31+（Ubuntu 20.04、Debian 11、Fedora 34） | Vulkan ローダ（`libvulkan1` / `vulkan-loader`） |
| macOS | 13 Ventura、Apple Silicon | Metal（内蔵） |
