Paneflow ships native binaries for Linux and macOS. Pick the guide for
your OS below. Every install path takes under two minutes and ends with
a verify step so you know it worked.

## Pick your platform

- [Linux](/docs/installation/linux) - `.deb`, AppImage, or tarball. AppImage is the universal default.
- [macOS](/docs/installation/macos) - signed and notarized `.dmg`. Apple Silicon today; Intel and a Homebrew tap land later.

## What you get

A single `paneflow` binary plus a shell launcher. The default config
lives at `~/.config/paneflow/paneflow.json` on Linux and
`~/Library/Application Support/paneflow/paneflow.json` on macOS. No
services, no daemons, no background processes.

## Need help?

For Linux PATH issues, jump to the [Linux troubleshooting section](/docs/installation/linux#what-if-paneflow-is-not-found-in-my-path).
For Gatekeeper blocks on macOS, see the [macOS Gatekeeper section](/docs/installation/macos#what-if-macos-gatekeeper-blocks-the-app).
For any other issue, file a [GitHub issue](https://github.com/ArthurDEV44/paneflow/issues).
