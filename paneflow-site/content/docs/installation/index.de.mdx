Paneflow vertreibt native Binärdateien für Linux und macOS. Wähle
unten die Anleitung für dein OS. Jeder Installationspfad braucht
unter zwei Minuten und endet mit einem Verifikationsschritt, damit
du weißt, dass es geklappt hat.

## Wähle deine Plattform

- [Linux](/docs/installation/linux) - `.deb`, AppImage oder Tarball. Das AppImage ist der universelle Standard.
- [macOS](/docs/installation/macos) - signiertes und notarisiertes `.dmg`. Apple Silicon heute; Intel und ein Homebrew-Tap kommen später.

## Was du bekommst

Eine einzige `paneflow`-Binärdatei plus einen Shell-Launcher. Die
Standard-Konfiguration liegt unter
`~/.config/paneflow/paneflow.json` unter Linux und
`~/Library/Application Support/paneflow/paneflow.json` unter macOS.
Keine Dienste, keine Daemons, keine Hintergrundprozesse.

## Brauchst du Hilfe?

Für PATH-Probleme unter Linux springe direkt zum [Linux-Abschnitt zur Fehlerbehebung](/docs/installation/linux#what-if-paneflow-is-not-found-in-my-path).
Für Gatekeeper-Blockierungen unter macOS siehe den [macOS-Gatekeeper-Abschnitt](/docs/installation/macos#what-if-macos-gatekeeper-blocks-the-app).
Bei jedem anderen Problem öffne ein [GitHub-Issue](https://github.com/ArthurDEV44/paneflow/issues).
