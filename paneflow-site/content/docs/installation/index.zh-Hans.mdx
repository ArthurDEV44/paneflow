Paneflow 为 Linux 和 macOS 提供原生二进制文件。在下方选择适用于你的
OS 的指南。每条安装路径耗时不到两分钟，最后都有一个验证步骤，让你
知道是否成功。

## 选择你的平台

- [Linux](/docs/installation/linux) - `.deb`、AppImage 或 tarball。AppImage 是通用默认选项。
- [macOS](/docs/installation/macos) - 已签名并公证的 `.dmg`。目前仅 Apple Silicon；Intel 和 Homebrew tap 将稍后推出。

## 你能得到什么

一个单独的 `paneflow` 二进制文件加一个 shell 启动器。默认配置位于
Linux 上的 `~/.config/paneflow/paneflow.json` 和 macOS 上的
`~/Library/Application Support/paneflow/paneflow.json`。没有服务、
没有守护进程、没有后台进程。

## 需要帮助？

针对 Linux 上的 PATH 问题，直接跳到 [Linux 故障排除部分](/docs/installation/linux#what-if-paneflow-is-not-found-in-my-path)。
针对 macOS 上的 Gatekeeper 拦截，参见 [macOS Gatekeeper 部分](/docs/installation/macos#what-if-macos-gatekeeper-blocks-the-app)。
其他任何问题，请提交 [GitHub issue](https://github.com/ArthurDEV44/paneflow/issues)。
