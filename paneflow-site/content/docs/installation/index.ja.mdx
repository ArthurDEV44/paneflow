Paneflow は Linux と macOS 向けのネイティブ バイナリを配布して
います。下からあなたの OS に対応するガイドを選んでください。
どのインストール パスも 2 分以内で完了し、最後に動作確認ステップ
があるので、成功したかどうかが分かります。

## プラットフォームを選択

- [Linux](/docs/installation/linux) - `.deb`、AppImage、または tarball。AppImage はユニバーサルなデフォルトです。
- [macOS](/docs/installation/macos) - 署名済みかつ公証済みの `.dmg`。現在は Apple Silicon のみ。Intel と Homebrew tap は今後追加予定です。

## 得られるもの

単一の `paneflow` バイナリとシェル ランチャーです。デフォルト設定
は Linux では `~/.config/paneflow/paneflow.json` に、macOS では
`~/Library/Application Support/paneflow/paneflow.json` に置かれます。
サービスもデーモンもバックグラウンド プロセスもありません。

## ヘルプが必要ですか？

Linux の PATH に関する問題は、[Linux トラブルシューティング セクション](/docs/installation/linux#what-if-paneflow-is-not-found-in-my-path)
へ進んでください。
macOS の Gatekeeper によるブロックについては、[macOS Gatekeeper セクション](/docs/installation/macos#what-if-macos-gatekeeper-blocks-the-app)
を参照してください。
その他の問題については、[GitHub issue](https://github.com/ArthurDEV44/paneflow/issues)
を作成してください。
