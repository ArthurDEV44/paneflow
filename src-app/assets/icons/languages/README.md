# Language icon SVGs

Drop language and file-type SVGs here.

Use lowercase kebab-case filenames that match `manifest.json`, for example:

- `rust.svg`
- `typescript.svg`
- `python.svg`
- `markdown.svg`
- `docker.svg`

Keep the SVGs monochrome or self-contained color SVGs. The diff UI can then map
file extensions to these stable filenames without scattering language-specific
logic through the renderer.
